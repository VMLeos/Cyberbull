Configurar y particionar SSD a mano para instalar Windows 7

En un equipo con Windows© 8 o 10 se conecta el SSD en que se instalará W7 se inicia como si fueramos a reparar el sistema, y seguir los pasos que se enumeran a continuación:

  1. Seleccionar idioma
  2. Reparar el equipo
  3. Windows no va a encontrar imágen, cancelar
  4. Opciones de recuperación --}simbolo del sistema
  5. X:\Sources }diskpart
  6. DISKPART} list disk
  7. DISKPART} sel disk 1 (el SSD)
  8. DISKPART} clean
  9. DISKPART} create part primary size=100
  10. DISKPART} format fs=ntfs quick
  11. DISKPART} assign letter =d
  12. DISKPART} create part primary size=5000
  13. DISKPART} format fs=ntfs quick
  14. DISKPART} assign letter =c
  15. DISKPART} exit
  16. X:\Sources} exit
  17. Reiniciar el equipo
  18. Elegir idioma de instalación
  19. Personalizar región, etc.
  20. Seleccionar partición primaria principal (50000 mb o sea 48 gb aprox)

Controlador NVMe de cliente para SSD Intel®

Este controlador de Windows® se puede utilizar para permitir que el sistema reconozca lo siguiente:

  1. Intel® SSD serie 665p
  2. Intel® SSD serie 660p
  3. SSD Intel® Optane™ serie 900P
  4. SSD Intel® Optane™ serie 905P
  5. Intel® SSD serie 750
  6. Intel® SSD Pro 7600p/760p/E 6100p Series
